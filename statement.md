PROJECT STATEMENT 
Project Title
Mess Demand Predictor

INTRODUCTION:
Managing food preparation in a college mess is often a challenge due to unpredictable 
student attendance, leading to food wastages or shortages. This project presents an 
intelligent solution that leverages Machine Learning to predict daily mess demand based 
on factors such as day of the week and menu type.

OBJECTIVE:
The primary objectives of this project are:
1. To reduce food wastage and improve resource planning
2. To implement a structured data storage system using DBMS
3. To create a system that learns a improves with new data

FUNCTIONAL MODULES:
Input Module 
1. Accepts user inputs such as day of the week and menu type
2. Takes real-time data to stimulate dynamic conditions
3. Allows entry of actual attendance for continuous learning

Output Module
1. Displays predicted mess demand using the trained ML model
2. Provides clear numerical output for decision- making
3. Help in planning food quantity and reducing wastage 

WORKFLOW:
1. The system will initialze and connect to the database.
2. Historical mess data will be retrieved.
3. The trained model predicts expected attendance

KEY HIGHLIGHTS:
1. Combines AI, DBMS, and CPS concpets in a single project
2. Real time input stimulation reflecting real-world scenarios
3. Practical application in reducing food wastage
